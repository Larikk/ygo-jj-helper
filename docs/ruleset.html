Ruleset

Click here to go back to the main page. Standard TCG rules apply unless stated otherwise:
  • A custom Banlist is used
  • Tribute Summons require no tributes
  • Quick-Play Spells are treated as Normal Spells
  • Only 1 Spell can be activated each turn
  • Only 1 Trap can be activated each turn
  • Only 1 Spell/Trap can be manually set each turn
  • No Ignition Effect Priority
  • No hand size limit
  • No first turn draw
