Ruleset

Click here to go back to the main page. Standard TCG rules apply unless stated otherwise:
  • A custom Banlist is used
  • Tribute Summons require no Tributes and are treated as Normal Summons
  • Quick-Play Spells are Spell Speed 1 (but are not treated as Normal Spells)
  • Only 1 Spell can be activated each turn
  • Only 1 Trap can be activated each turn
  • Only 1 Spell/Trap can be manually set each turn
  • No Ignition Effect Priority
  • No hand size limit
  • No first turn draw

Clarifications on individual rules

  • If the activation of your Spell/Trap is negated, you can attempt to activate another one on the same turn. This is not the case if the effect is negated. (Examples: Solemn Judgement negates activation, Imperial Order negates effect)
  • Tribute Summoning
    • Treating Tribute Summons as a Normal Summon is not optional; You can not tribute monsters to Normal Summon a high-level monster unless its card text allows or requires that.
    • Monster with an effect that triggers on their own Tribute Summon can not active this effect when they are Normal Summoned (Example: Monarchs).
    • Monster with an effect that allows them to be Tribute Summoned with more Tributes than necessary can either be Normal Summoned without this effect or Tribute Summoned with this effect by tributing the specified amount of monsters (Example: Moisture Creature).
    • Monster with restrictions on their Tribute Summon regarding amount and type of Tributes can only be Tribute Summoned by tributing the necessary Tributes (that means Normal Summoning without Tributes is not allowed.)
